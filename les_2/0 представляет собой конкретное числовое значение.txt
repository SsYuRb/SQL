0 представляет собой конкретное числовое значение, которое равно нулю. Оно может быть использовано в вычислениях и операциях, и имеет определенное значение.NULL означает отсутствие значения. NULL используется для обозначения отсутствия информации или недоступности значения, и может быть присвоено любому типу данных в SQL.Сравнение значения с NULL с помощью оператора "=" вернет значение NULL, а не истину или ложь. Для сравнения значений с NULL нужно использовать операторы "IS NULL" или "IS NOT NULL".
